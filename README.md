# Welcome to Agartha, a land down under

## Requirements
- Download and install Minecraft Launcher
- Download and install CurseForge

## Installation
1. Download Agartha-X.X.X.zip compressed file.
2. Open CurseForge and select Minecraft.
3. Go to My Modpacks and click on Import.
4. Choose the Agartha-X.X.X.zip file.
5. Once Agartha modpack is loaded, click Play (NOT arrow beside Play button) to launch Agartha modpack using Minecraft launcher.

## Mods
Most mods are rendered on the server. Client side mods are optional (QOL).

### Server
- Better Advancements: enhanced achievent trees.
- Biomes O' Plenty: 50 new biomes.
- Clumps: clumps XP orbs together to reduce lag (disable for end dragon?).
- Comforts: bed portability (sleeping bags and hammocks).
- Dynamic Lights: holding a torch will illuminate surrounding area dynamically.
- Just Enough Items: item list/recipes on inventory/crafting bench like Bedrock.
- Traveler's Backpack: adds upgradeable backpacks for extra inventory.
- Tree Harvester: crouch+axe to chop a tree in 1 pull (takes longer the more logs).
- Waystones: craftable blocks you can teleport to.
- Xaero's Minimap + World Map: permanent minimap HUD + big world map (with discovered aread only).
- Your Items Are Safe: chest + armor rack with your shit when you die (Aldo WILL steal if alive and nearby).
- Enhanced Structures: such as desert temples, dingeons, jungle temples, mineshafts, witch huts, and random bridges between landmasses.

### Client (recommended, not required)
- Better F3: enhanced and customizeable F3 menu.
