# Welcome to Agartha, a land down under

## Requirements

- Download and install Minecraft Launcher
- Download and install CurseForge

## Installation

1. Download Agartha-X.X.X.zip compressed file.
2. Open CurseForge and select Minecraft.
3. Go to My Modpacks and click on Import.
4. Choose the Agartha-X.X.X.zip file.
5. Once Agartha modpack is loaded, click Play (NOT arrow beside Play button) to launch Agartha modpack using Minecraft launcher.
